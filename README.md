# Erebus
Discord <-> Xonotic
## Features
- Link a Discord channel to a Xonotic server
- Bi-directional chat relaying
- End-Match scoreboard
- Commands
  - `!status`
  - `!xonstat <id>`
  - `!rcon <cmd>` (for owner only)
## Advanced Features
- Adding tracks from YouTube ingame to SMB Mod Radio (needs `web/radio.cgi`)
